About the Project

Data Visualization on live site using Flask framework using live data from GOOGLE.

This web app is developed for embedding Bokeh charts in a live website.

The Data is stored in pandas dataframe for easy access and management

The Finance data is fetched from google. And Converted into representable form i.e. "Candlestick chart"
